# 智慧零用金報帳系統 (Smart Petty Cash System)

這是一套基於網頁技術的現代化報帳系統，無需安裝複雜環境即可使用。

## 功能特色
- **業務端**: QR Code 發票掃描、自動分類、即時小計。
- **會計端**: 審核、員工管理、匯出 CSV 報表、關鍵字設定。
- **免安裝**: 使用瀏覽器即可運行。

## 如何使用

### 1. 啟動系統
由於系統使用了 ES Modules，建議透過本地伺服器開啟以獲得最佳體驗 (直接點擊 index.html 可能會有瀏覽器安全限制)。

**Windows (已安裝 Python):**
1. 在資料夾空白處按右鍵，選擇「在終端機開啟」 (Powershell / cmd)
2. 輸入指令:
   ```bash
   python -m http.server
   ```
3. 開啟瀏覽器訪問: `http://localhost:8000`

### 2. 預設帳號
系統內建展示資料與使用者切換功能 (位於右上角):

- **管理員 (Admin)**
  - 權限: 審核報帳、管理員工、設定關鍵字、匯出所有報表。
  - 用於: 會計/財務人員。

- **員工 (John Doe)**
  - 權限: 掃描/新增報帳、查看個人紀錄。
  - 用於: 業務/一般員工。

### 3. 操作說明
- **新增報帳**: 點擊「掃描發票」或「手動輸入」。
- **代填報帳**: 登入管理員帳號 -> 員工管理 -> 點擊「代填報帳單」。
- **匯出報表**: 
  - 全部匯出: 審核頁面 -> 「匯出全部 CSV」
  - 個人匯出: 員工管理 -> 點擊員工旁的「匯出 CSV」

## 系統架構
- `index.html`: 入口網頁
- `js/`: 核心邏輯
- `css/`: 樣式表 (Premium Blue Design)
- LocalStorage: 資料儲存於瀏覽器本地，關閉視窗後資料仍會保留 (清除瀏覽器快取會重置)。
