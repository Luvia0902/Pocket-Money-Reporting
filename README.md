# 智慧零用金報帳系統 (Smart Petty Cash System)

這是一套基於網頁技術的現代化報帳系統，支援雲端部署 (GitHub Pages)，員工可使用手機隨時隨地報帳。

## 功能特色
### 業務端 (前端)
- **智慧掃描**: QR Code 發票掃描 (需在 HTTPS 或 Localhost 環境)。
- **自動分類**: 內建 50+ 組關鍵字，自動將商家歸類為「交通費、伙食費、文具用品、郵電費、運費」等類別。
- **規則查詢**: 設有獨立「規則說明」頁面，可以標籤雲方式檢視所有對應規則。

### 會計端 (後台)
- **審核機制**: 一鍵核准員工報帳申請。
- **員工管理**: 
  - 可直接新增員工帳號 (免 Email 邀請)。
  - **代填報帳**: 管理員可協助員工代為輸入報帳資料。
- **進階匯出**: 
  - 支援 **日期範圍篩選** (本月、上月、或自訂日期區間)。
  - 可匯出「全部資料」或「特定員工」的 CSV 報表。
- **規則管理**: 管理員可新增/刪除關鍵字對應規則 (支援類別下拉選單)。

## 如何使用

### 線上使用 (推薦)
本系統已支援 GitHub Pages，請直接使用管理員提供的網址即可連線使用 (手機/電腦皆可)。

### 本地開發/測試
若需在個人電腦上離線執行：
1. 點擊 `start_system.bat` (Windows)。
2. 或手動執行 `python -m http.server`。
3. 瀏覽器開啟 `http://localhost:8000`。

## 帳號權限說明

系統內建使用者切換模擬功能 (位於右上角選單):

- **管理員 (Admin)**
  - 權限: 最高權限，可審核、管理員工、修改規則、匯出報表。
  - 特有功能:「代填報帳單」、「匯出範圍設定」。

- **員工 (John Doe)**
  - 權限: 僅限新增個人報帳、查看個人紀錄。
  - 可檢視「規則說明」頁面但無法編輯。

## 操作指南

### 1. 匯出報表
在「審核報帳」或「員工管理」頁面，利用上方的 **匯出範圍** 工具列：
- 選擇「本月/上個月」或「自訂範圍」設定日期。
- 點擊「匯出全部 CSV」或個別員工的「匯出 CSV」，系統將只產出該時段的資料。

### 2. 管理關鍵字
- 前往「規則說明」頁面。
- 規則已依照 **交通、伙食、文具、郵電、運費** 自動分類顯示。
- 管理員可點擊標籤上的 `x` 刪除，或填寫上方表單新增規則。

## 系統分類標準 (預設)
系統預設包含以下會計科目自動對應：
- **交通費**: 加油站、停車場、Uber、高鐵...
- **伙食費**: 餐廳、各類速食、便當店...
- **文具用品**: 書局、辦公用品店...
- **郵電費**: 電信公司、郵局...
- **運費**: 宅配、快遞...
