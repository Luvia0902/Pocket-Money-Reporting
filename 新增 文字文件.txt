雲端資料庫升級完成 (Firebase)
我已經完成了系統升級，現在系統支援「跨裝置資料同步」。

接下來請您依照以下步驟啟用 (只需設定一次)：
申請免費資料庫：
前往 Firebase Console。
點擊「新增專案」 -> 輸入專案名稱 (例如 ZeroMoney-DB) -> 繼續 -> 繼續 -> 建立專案。
進入專案後，點擊左側選單的 「建置」 (Build) -> 「Firestore Database」。
點擊 「建立資料庫」 -> 選擇 「以測試模式啟動 (Start in test mode)」 (這很重要，讓您無需繁瑣設定即可讀寫) -> 下一步 -> 選擇位置 (可選 asia-east1 台灣或預設) -> 啟用。
取得設定檔 (Config)：
點擊左上角的 「專案總覽」 (Project Overview) 旁邊的齒輪圖示 -> 「專案設定」。
在下方「您的應用程式」區域，點擊 </> (Web) 圖示。
輸入暱稱 (例如 Web) -> 註冊應用程式。
您會看到一段 const firebaseConfig = { ... }; 的程式碼。請複製 大括號 { ... } 內的所有內容 (包含大括號)。
啟用連線：
開啟您的 ZeroMoney 網頁 (電腦或手機皆可，建議先用電腦設定)。
登入 admin@gmail.com 或 assistant@company.com。
進入 「管理後台」 -> 點擊新出現的 「雲端設定」 分頁。
將剛剛複製的 { ... } 內容貼上，點擊 「儲存並連線」。
一旦設定完成，系統會自動重新啟動並開始同步。之後您在手機上只需做同樣的步驟 (貼上設定)，兩邊資料就會互通了！
